# Assumptions Register Template

Back to workspace landing: https://www.notion.so/Forge-Data-Center-Open-Source-Public-Workspace-3324a1784e5980bc87b8fc9c32645c96

## Purpose

Use this register to keep every major planning assumption explicit, owned, and reviewable.

## Suggested Fields

- Assumption ID
- Domain (Site, Power, Cooling, Network, Runtime, Orchestration, Economics, Operations)
- Assumption statement
- Source type (Internal formula, Artifact coefficient, External benchmark, Expert judgment)
- Source link or file anchor
- Current value
- Confidence (Low / Medium / High)
- Owner
- Last reviewed date
- Next review date
- If wrong, what breaks?
- Mitigation plan

## Review Cadence

- Weekly during active planning
- Before any executive decision memo
- Immediately after coefficient or model-version updates
