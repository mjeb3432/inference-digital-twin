# Scenario Intake Worksheet Template

Back to workspace landing: https://www.notion.so/Forge-Data-Center-Open-Source-Public-Workspace-3324a1784e5980bc87b8fc9c32645c96

## Purpose

Standardize scenario design before running Forge or Digital Twin simulations.

## Intake Blocks

### 1) Objective

- Primary objective (latency-first, cost-first, capacity-first, resilience-first)
- Decision deadline
- Decision owner

### 2) Workload Profile

- Workload class
- Prompt/completion shape
- Steady and burst demand
- SLO target

### 3) Hard Constraints

- Budget envelope
- Maximum available power
- Site or region constraints
- Compliance/security constraints

### 4) Candidate Scenarios

- Baseline scenario
- Alternative A
- Alternative B
- What variable is intentionally changed in each comparison?

### 5) Success Criteria

- Must-hit metrics
- Nice-to-have metrics
- Risk thresholds

## Minimum Rule

A scenario should not be promoted for review unless objective, constraints, and comparison logic are fully defined.
