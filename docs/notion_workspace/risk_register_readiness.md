# Risk Register and Readiness Checklist

Back to workspace landing: https://www.notion.so/Forge-Data-Center-Open-Source-Public-Workspace-3324a1784e5980bc87b8fc9c32645c96

## Risk Register Fields

- Risk ID
- Risk type (Model / Data / Operational / Program / Vendor)
- Description
- Trigger condition
- Detection signal
- Likelihood
- Impact
- Owner
- Mitigation
- Status

## Readiness Checklist

- Scenario schema validation passes
- Calibration artifact compatibility passes
- Phase-gate validation passes
- Report provenance fields present
- Limitations reviewed and accepted
- Comparison set includes a true baseline
- Decision memo completed
- Revisit trigger and owner defined

## Exit Rule

No scenario should be marked decision-ready unless every readiness item is complete.
