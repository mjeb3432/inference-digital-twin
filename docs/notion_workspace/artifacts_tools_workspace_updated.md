## Suggested Operating Sequence

1. Use Scenario Intake to define objective and constraints.
2. Log major assumptions in the Assumptions Register.
3. Run baseline + alternatives, then use Sensitivity Sweep to test uncertainty.
4. Capture recommendation in the Decision Memo.
5. Validate readiness with Risk/Register checklist before committing.

## Why This Matters

These artifacts create a repeatable planning workflow. They reduce undocumented judgment calls and make it easier for cross-functional teams to audit, challenge, and improve decisions over time.
