## What Scenario Matrix v1 Is

Scenario Matrix v1 is the decision framework that structures planning into gated phases.

## How to Use It

1. Start with a baseline scenario
2. Complete decisions phase by phase
3. Resolve validation errors before phase confirm
4. Export scenario snapshot after each major milestone
5. Compare scenarios only when they are structurally complete

## Suggested Scenario Set

- Baseline (conservative build)
- Performance-first variant
- Cost-first variant
- Reliability-first variant

## Comparison Method

Keep all assumptions fixed except the strategic variable under test. This creates interpretable deltas and avoids mixed-cause conclusions.

## Reporting Standard

For each scenario comparison, capture:

- objective
- assumptions changed
- metrics moved
- risk implications
- recommendation

Use the Decision Memo artifact page to standardize this.
